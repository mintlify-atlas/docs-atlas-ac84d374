# Introduction

Ritual is a 4chan archiver that focuses on simplicity and flexibility. Built with Python 3.12+, it provides powerful filtering capabilities to archive exactly what you need while avoiding unnecessary downloads.

## What is Ritual?

Ritual allows you to archive 4chan threads and media with fine-grained control over what gets saved. Whether you want to archive everything from a board, only specific threads matching certain keywords, or just the text without media, Ritual gives you complete control.

The archiver runs synchronously in a step-by-step manner that's easy to read and debug, making it accessible for developers who want to understand or modify its behavior.

## How it works

Ritual operates in a continuous loop, processing each configured board sequentially:

1. **Fetch catalog** - Retrieves the current catalog of threads from the board
2. **Filter threads** - Applies your whitelist/blacklist rules to determine which threads to archive
3. **Fetch posts** - Downloads thread data for modified or new threads
4. **Save text** - Stores post text and metadata in the database using the Asagi schema
5. **Download media** - Selectively downloads thumbnails and full media based on your configuration
6. **Repeat** - Waits for the configured cooldown period and starts the next loop

The archiver maintains state between runs, tracking which threads have been modified and which media has already been downloaded. This ensures efficient operation and avoids duplicate work.

<Note>
Ritual runs synchronously and processes boards one at a time. This design choice makes the code easy to read and debug while still being efficient enough for most archiving needs.
</Note>

## Database schema

Ritual uses the [Asagi schema](https://github.com/sky-cake/asagi-tables), which is the de facto standard for 4chan archives. This ensures compatibility with existing tools and makes it easy to migrate data or use third-party analysis tools.

The schema includes tables for:
- Thread and post data
- Media metadata and hashes
- Board information
- Deleted thread tracking
